🧹 WinSxS Cleanup Tool (Windows Component Store Cleanup)

Windows의 WinSxS(Component Store)를 정리하는
PowerShell 기반 GUI 유틸리티입니다.

✔ DISM 기반 정식 Windows 명령어만 사용
✔ 관리자 권한 필요 (시스템 정리 특성상)
✔ 로그 출력 + 용량 절감 수치 표시
✔ ResetBase 옵션 포함 (되돌릴 수 없음)
✔ 다크 모드 UI 지원

📌 개인이 직접 제작한 도구이며,
불필요한 백그라운드 동작이나 네트워크 통신은 없습니다. (스크립트는 ChatGPT의 도움을 받아 작성하였습니다)

※ 중요 안내 (백신 오탐 관련)

본 프로그램은 PowerShell 스크립트를 EXE로 패키징한 도구입니다.
이러한 형태의 시스템 관리 유틸은 일부 백신 엔진에서
오탐(false positive)으로 탐지될 수 있습니다.

VirusTotal 기준:
- 다수의 메이저 백신 엔진에서는 정상(Undetected)
- 일부 휴리스틱 / 머신러닝 기반 엔진에서만 탐지됨

이는 실제 악성 행위가 아닌,
"관리자 권한 + 시스템 명령 실행" 특성에 따른 오탐입니다.

✔ 네트워크 통신 없음
✔ 파일 다운로드/업로드 없음
✔ 레지스트리 상주 없음
✔ 자기 복제 없음

Windows 내장 도구(DISM)만을 사용하며
모든 동작은 UI 로그 창에 표시됩니다.

스크립트 기반 도구 특성상 불안하신 분은 사용하지 않으셔도 됩니다.

문제 있으면 바로 제보 주세요. 바로 확인합니다.
문제 발생 시 스크린샷과 함께 제보해주시면 빠르게 확인하겠습니다.

.ps1 파일은 실행 정책에 따라 바로 실행되지 않을 수 있으며, 검토용으로 제공됩니다.
