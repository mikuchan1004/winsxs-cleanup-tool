# 🧹 WinSxS Cleanup Tool

Windows의 Component Store(WinSxS) 폴더를  
**안전하게 분석하고 정리할 수 있는 GUI 기반 유틸리티**입니다.

DISM 명령을 기반으로 동작하며,  
관리자 권한 확인 · 분석 결과 안내 · 예상 절감 용량 표시 등  
일반 사용자도 비교적 안심하고 사용할 수 있도록 설계되었습니다.

---

## ✨ 주요 기능

- DISM 기반 WinSxS 구성 요소 분석
- 예상 절감 용량(Reclaimable Size) 표시
- 분석 결과에 따라 정리 버튼 자동 활성 / 비활성
- ResetBase 옵션 지원 (⚠ 되돌릴 수 없음)
- 진행률 바 + 예상 남은 시간 표시
- 한글 UI 및 한글 로그 출력
- 관리자 권한 / OS 환경 자동 체크

---

## 🖥 지원 운영체제

| 운영체제 | 지원 여부 |
|---------|----------|
| Windows 7 | ❌ 미지원 |
| Windows 8 / 8.1 | ❌ 미지원 |
| Windows 10 | ✅ 지원 |
| Windows 11 | ✅ 지원 |

※ DISM `/AnalyzeComponentStore` 명령을 지원하는 환경에서만 정상 동작합니다.

## 📦 제공 파일

- **WinSxS_Cleanup_Tool.exe**  
  (PowerShell 스크립트를 기반으로 빌드된 실행 파일)

- **WinSxS_Cleanup.zip**  
  (exe + 안내 문서 포함)

👉 일반 사용자에게는 **ZIP 파일 다운로드를 권장**합니다.

## ▶ 사용 방법

1. `WinSxS_Cleanup_Tool.exe`를 **관리자 권한으로 실행**
2. **[Analyze(분석)] 버튼 클릭**
3. 분석 완료 후 표시되는 **예상 절감 용량 확인**
4. 필요 시 **[Start Cleanup(정리)] 버튼 클릭**
5. 정리 완료까지 대기

---

## 📊 분석 결과 안내

- **예상 절감 용량: 0 MB**
  - 현재 정리 가능한 WinSxS 구성 요소가 없는 상태입니다.
  - 이 경우 정리 버튼은 비활성화됩니다.

- 분석에는 **약 1~5분** 정도 소요될 수 있습니다.
- 분석 중에는 창이 잠시 멈춘 것처럼 보일 수 있으나 정상 동작입니다.

---

## ⚠ ResetBase 옵션 주의

- ResetBase 옵션을 사용하면:
  - 기존 Windows 업데이트를 **되돌릴 수 없습니다**
  - 향후 업데이트 제거가 불가능해질 수 있습니다

👉 충분히 이해한 경우에만 사용하세요.

---

## ⚠ 주의 사항

- 반드시 **관리자 권한**으로 실행해야 합니다
- 중요한 작업 전에는 **시스템 복원 지점 생성 권장**
- 정리/분석 중 강제 종료는 권장하지 않습니다
