# Security Policy

## Supported Versions
현재 최신 릴리즈(“Latest”) 버전을 우선 지원합니다.

## Reporting a Vulnerability
보안 취약점/악용 가능성이 의심되는 문제는 공개 이슈로 올리지 말고 아래 방식으로 제보해주세요.

### ✅ 권장: GitHub Private Vulnerability Reporting
- 저장소 → **Security** → **Advisories** → **Report a vulnerability**

### ✉ 대안: 비공개 연락
Private Vulnerability Reporting이 불가능한 경우, 아래 정보와 함께 비공개로 제보해주세요.
- 사용 버전 (예: v1.0.5)
- 재현 방법 / PoC (가능하면)
- 영향 범위(권한 상승, 데이터 노출 등)
- 관련 로그/스크린샷 (개인정보/민감정보는 제거)

## Disclosure
제보 확인 후 가능한 빠르게 triage하고, 패치 및 릴리즈 계획을 공유합니다.
