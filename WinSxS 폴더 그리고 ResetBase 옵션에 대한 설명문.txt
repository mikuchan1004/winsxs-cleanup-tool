[ WinSxS (Windows Component Store)란? ]

WinSxS 폴더는 Windows의 핵심 시스템 구성 요소(Component Store)를 저장하는 공간입니다.
Windows 업데이트, 기능 추가/제거, 시스템 복구 및 롤백 시 사용됩니다.

이 폴더에 저장된 파일들은 중복으로 보일 수 있으나,
실제로는 하드 링크(Hard Link) 구조를 사용하여
파일이 여러 위치에서 참조되고 있을 수 있습니다.

탐색기에서 보이는 용량과 실제 디스크 사용량은 다를 수 있습니다.

[ WinSxS를 직접 삭제하면 안 되는 이유 ]

- Windows 업데이트 오류
- 시스템 파일 손상
- 기능 복구/롤백 불가
- 최악의 경우 부팅 불가

이러한 문제가 발생할 수 있으므로
WinSxS 폴더는 직접 삭제해서는 안 됩니다.

[ 이 도구는 무엇을 하나요? ]

본 도구는 WinSxS 폴더를 직접 삭제하지 않으며,
Windows에서 공식적으로 제공하는 DISM 명령어를 사용해
불필요한 구성 요소를 안전하게 정리합니다.

[ ResetBase 옵션 주의 ]

ResetBase 옵션을 사용하면:
- 이전 Windows 업데이트로 되돌릴 수 없게 됩니다
- 현재 설치된 업데이트 상태를 기준으로 고정됩니다

문제 발생 시 롤백이 불가능하므로
필요성과 위험을 이해한 경우에만 사용하시기 바랍니다.
