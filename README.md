# WinSxS Cleanup Tool

🧹 Windows의 Component Store(WinSxS) 폴더를 안전하게 정리하기 위한  
**GUI 기반 유틸리티**입니다.

DISM 명령을 기반으로 동작하며, 관리자 권한 확인 및  
주의 사항 안내를 포함하여 비교적 안전하게 사용할 수 있도록 설계되었습니다.

---

## ✨ 주요 기능
- DISM을 이용한 WinSxS 정리
- ResetBase 옵션 지원
- 관리자 권한 자동 확인
- 직관적인 GUI 인터페이스
- 실행 결과 안내 및 오류 처리

---

## 📦 제공 파일
- `WinSxS_Cleanup_Tool.exe`  
  (PowerShell 스크립트를 기반으로 빌드된 실행 파일)
- `WinSxS_Cleanup_Tool_v1.0.0.zip`  
  (exe + 안내 문서가 포함된 압축 파일)

👉 **권장:** ZIP 파일 다운로드 후 사용

---

## ▶ 사용 방법
1. `WinSxS_Cleanup_Tool.exe`를 **관리자 권한으로 실행**
2. 안내 문구 확인 후 옵션 선택
3. 정리 작업 완료까지 대기

---

## ⚠️ 주의 사항
- 관리자 권한이 필요합니다.
- **ResetBase 옵션은 적용 후 되돌릴 수 없습니다.**
- Windows 10 / 11 환경에서 사용을 권장합니다.
- 중요한 작업 전 **시스템 복원 지점 생성**을 권장합니다.
